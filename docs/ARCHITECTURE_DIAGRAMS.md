# Architecture diagrams (AWS icons)
This project uses a manual Draw.io diagram for architecture visuals.
## Files
- As-is (implemented): `docs/architecture.drawio`
- Target (planned): `docs/architecture-target.drawio`
## Edit in VS Code
Install the **Draw.io Integration** extension, then open `docs/architecture.drawio`.
## Export
Use the Draw.io UI to export PNG/SVG when needed.
